SD 分镜 Prompt 管理器

{
  "type": "sd-shot-list",
  "version": 2,
  "exportedAt": "ISO8601 时间",
  "items": [
    { "title": "分镜 1", "positive": "...", "negative": "..." },
    { "title": "分镜 2", "positive": "...", "negative": "..." }
  ]
}

兼容导入旧版 v1（将旧字段 prompt 自动映射为 positive）。
